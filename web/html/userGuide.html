U.S. Department of Commerce

Enforcement and Compliance (E&C) Mapping Software


User Guide:


  • Data Category Form

    Only TANC cases is enabled as of now. The other 4 options are disabled but the flow for those are pretty similar to TANC(the corresponding filter forms for those are developed in the same way)

    Select 'TANC Casework' option and hit continue – to move on to the file upload form.



  • File Upload Form:

    The file upload form requires you to browse and select the Excel file with the data. Assumptions involved: The data is found in a continuous table with valid entries in the rows and columns.

    Select an Excel file (Only files with .xls and .xlsx extensions will be accepted). Click on 'Upload'.



    Select Yes in the dialog box asking for permission to run ActiveX control – This is required to read the input Excel file data into the backend.



    Click on 'Continue' after the loading is Complete.

  • Filter Form:

    Select all the required filters for which you wish to see the data on map.

    Click on 'Continue' located at the bottom of the this page.



  • Map Display Type:

    Select a display option and click on 'Continue'.



  • Map:

    Maps generated are interactive. If you hover over a particular region it would give you the details for that region.

    Also, below you will find a table summarizing the information.
